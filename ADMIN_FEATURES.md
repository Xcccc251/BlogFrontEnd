# 后台管理功能说明

## 概述

已在博客前台集成了后台管理功能，包括文章管理和标签管理。采用了与 publish 和 graph 页面相同的浅蓝色（#F0F8FF）背景，提供更一致的用户体验。

## 功能特点

### 1. 分页功能
- **前端分页**：无需后端支持，加载全部数据后在前端进行分页
- **灵活配置**：支持 10、20、30、50、100 条/页
- **完整功能**：包含总数显示、页码跳转、页大小切换
- **搜索保持**：搜索后自动重置到第一页，保持分页状态

### 2. 界面设计
- **浅蓝色背景**：与 publish 和 graph 页面保持一致的 #F0F8FF 背景色
- **现代化侧边栏**：
  - 左侧固定侧边栏，宽度 200px
  - 菜单项带有圆角和渐变背景
  - 激活菜单项有蓝色左侧边框指示器
  - 悬停效果流畅自然
  - 阴影效果提升层次感
- **卡片式布局**：所有内容区域使用卡片布局，带有圆角和阴影
- **响应式设计**：在小屏幕上侧边栏自动缩小为图标模式

### 3. 文章管理 (/admin/article)

#### 搜索功能
- 标题搜索
- 分类筛选
- 标签筛选
- 状态筛选（公开/私密/草稿）
- 置顶筛选

#### 列表展示
- 文章编号
- 封面图预览（可点击放大）
- 标题（超长显示省略号）
- 类型标签（原创/转载/翻译）
- 分类标签
- 标签列表（超过2个显示+N）
- 状态下拉选择（可直接修改）
- 置顶开关（可直接切换）
- 访问量统计
- 作者信息
- 发布时间
- 斑马纹表格（提升可读性）

#### 分页功能
- 显示总记录数
- 每页显示数量可选：10、20、30、50、100
- 上一页/下一页按钮
- 页码快速跳转
- 支持前端分页（无需后端支持）

#### 操作功能
- **查看**：在新标签页打开文章详情
- **编辑**：在新标签页打开编辑界面
- **删除**：支持单个或批量删除
- **状态修改**：直接在列表中修改文章状态
- **置顶设置**：直接在列表中设置是否置顶

### 4. 标签管理 (/admin/tag)

#### 搜索功能
- 标签名称搜索

#### 列表展示
- 标签编号
- 标签名称
- 关联文章数量
- 创建时间
- 更新时间
- 斑马纹表格（提升可读性）

#### 分页功能
- 显示总记录数
- 每页显示数量可选：10、20、30、50、100
- 上一页/下一页按钮
- 页码快速跳转
- 支持前端分页（无需后端支持）

#### 操作功能
- **新增标签**：弹窗表单，支持验证
- **编辑标签**：弹窗表单，回显数据
- **删除标签**：支持单个或批量删除
- **搜索标签**：按名称快速筛选

## 使用指南

### 访问管理后台

1. 确保已登录博客账号
2. 点击顶部菜单栏的"管理"按钮
3. 系统会自动跳转到文章管理页面
4. 可通过左侧侧边栏切换不同的管理功能

### 文章管理操作

#### 搜索文章
1. 在搜索表单中填写搜索条件
2. 点击"搜索"按钮
3. 点击"重置"按钮清空搜索条件

#### 修改文章状态
1. 找到要修改的文章
2. 直接点击"状态"列的下拉框
3. 选择新状态（公开/私密/草稿）
4. 系统自动保存

#### 设置文章置顶
1. 找到要置顶的文章
2. 点击"置顶"列的开关按钮
3. 系统自动保存

#### 删除文章
- **单个删除**：点击操作列的"删除"按钮
- **批量删除**：
  1. 勾选要删除的文章
  2. 点击顶部的"批量删除"按钮
  3. 确认删除操作

### 标签管理操作

#### 新增标签
1. 点击"新增标签"按钮
2. 在弹窗中输入标签名称（1-20个字符）
3. 点击"确定"保存

#### 编辑标签
1. 点击操作列的"编辑"按钮
2. 修改标签名称
3. 点击"确定"保存

#### 删除标签
- **单个删除**：点击操作列的"删除"按钮
- **批量删除**：
  1. 勾选要删除的标签
  2. 点击顶部的"批量删除"按钮
  3. 确认删除操作

## 样式特色

### 配色方案
- **主背景色**：#F0F8FF（浅蓝色）
- **主色调**：#409EFF（蓝色）
- **侧边栏背景**：白色
- **激活菜单背景**：渐变蓝色（#e6f7ff 到 #f0f9ff）
- **悬停背景**：#f5f7fa

### 阴影效果
- **侧边栏阴影**：`2px 0 8px rgba(0, 0, 0, 0.05)`
- **卡片阴影**：多层阴影提升层次感

### 圆角设计
- 菜单项：6px 圆角
- 卡片：8px 圆角
- 弹窗：8px 圆角

### 动画效果
- 所有交互都有 0.3s 的过渡动画
- 悬停和激活状态流畅自然
- 菜单切换平滑

## 响应式适配

### 桌面端（> 768px）
- 侧边栏宽度：200px
- 显示完整的菜单文字
- 激活指示器在左侧

### 移动端（≤ 768px）
- 侧边栏宽度：60px
- 只显示图标，隐藏文字
- 标题垂直显示
- 激活指示器在顶部

## 技术实现

### 前端技术栈
- Vue 3 Composition API
- TypeScript
- Element Plus UI 组件库
- SCSS 样式预处理器

### API 集成
- 复用 blog 项目现有的 API
- 统一的 HTTP 请求封装
- 统一的错误处理

### 路由配置
```typescript
{
  path: '/admin',
  component: () => import('@/views/Admin/index.vue'),
  name: 'admin',
  redirect: '/admin/article',
  meta: {
    title: '后台管理',
    requireAuth: true
  },
  children: [
    {
      path: 'article',
      name: 'adminArticle',
      meta: { title: '文章管理', requireAuth: true }
    },
    {
      path: 'tag',
      name: 'adminTag',
      meta: { title: '标签管理', requireAuth: true }
    }
  ]
}
```

3. **操作确认**：删除等危险操作都有二次确认

5. **响应式**：确保在各种屏幕尺寸下都能正常使用
